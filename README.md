# SimpleNNV
A sleek &amp; minimal neural network visualizer built with Three.js

> [**See it in action!**](https://jordaxn.github.io/SimpleNNV/) `v1.0.0`

# Overview
Most neural network diagrams are static and flat. SimpleNNV maps abstract architectures into a playable 3D space, allowing you to visualize data flow and topology in real time.

# Features
Dynamic Topology – Add/remove hidden layers and scale neuron density (1–12) on the fly.

Animated Propagation – Trigger a "Forward Pass" to see the signal pulse through weights.

Orbital View – Full 3D rotation with an optional "Drift" mode for cinematic visualization.

Performance – Efficient buffer-based rendering for smooth 60fps interaction.

# Tech Stack
Engine: Three.js (Octahedron geometries & dynamic line rendering)

Interface: Modern CSS Glassmorphism

Logic: Vanilla ES6+ JavaScript

# Quick Start
Adjust: Use + / − to modify layer counts.

Fire: Hit Forward pass → to animate the network.

View: Toggle Drift for auto-rotation or drag to explore manually.
